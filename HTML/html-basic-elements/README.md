# HTML Basics

## HyperText Markup Language

HTML stands for **HyperText Markup Language**.
It is used to create and structure web pages.
HTML tells the browser how the content should appear on the webpage.

---

# HTML Element

An **HTML element** is the basic building block used to create a web page.

An element is created by:

* a **start tag**
* the **content**
* an **end tag**

The content is placed between the start and end tag.

### Syntax

```
<tagname>content</tagname>
```

### Example

```
<p>This is a paragraph</p>
```

---

# HTML Tags

HTML **tags** are keywords used for a specific purpose to display and format content on a webpage.

Tags tell the browser what type of content it is.

### Example

```
<h1>This is a heading</h1>
```

---

# HTML Attributes

**Attributes** provide additional information about HTML elements.

Attributes are written inside the **opening tag** and help define or change the behavior of the element.

### Example

```
<a href="https://www.google.com/">Google</a>
```

Here `href` is an attribute that provides the link.

---

# Structure of HTML Code

A basic HTML page contains the following structure:

1. **Document type** – tells the browser that the document is HTML
2. **HTML root element** – contains the entire webpage
3. **Head section** – contains meta information
4. **Title** – name of the webpage shown in the browser tab
5. **Body section** – contains the visible content of the webpage

---

# Boilerplate Code

Boilerplate code is the **basic template of an HTML document**.

```
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>

</body>
</html>
```

## Heading Tags

Heading tags are used to define headings on a webpage.

`<h1>` is the largest heading and `<h6>` is the smallest heading.

### Example

```
<h1>Heading 1</h1>
<h2>Heading 2</h2>
```

---

## Paragraph Tag

The `<p>` tag is used to create paragraphs.

### Example

```
<p>This is a paragraph</p>
```

---

## Mark Tag

The `<mark>` tag is used to **highlight text**.

### Example

```
<mark>Highlighted text</mark>
```

---

## Strong Tag

The `<strong>` tag is used to show **important text in bold**.

### Example

```
<strong>Important text</strong>
```

---

## Bold Tag

The `<b>` tag is used to display text in **bold style**.

### Example

```
<b>Bold text</b>
```

---

## Em Tag

The `<em>` tag is used to show **emphasized text** and it usually appears in italic.

### Example

```
<em>Emphasized text</em>
```

---

## Italic Tag

The `<i>` tag is used to display text in **italic style**.

### Example

```
<i>Italic text</i>
```

---

## Underline Tag

The `<u>` tag is used to **underline text**.

### Example

```
<u>Underlined text</u>
```

---

## Abbreviation Tag

The `<abbr>` tag is used to show **abbreviations**.
When we hover on the text it shows the full meaning.

### Example

```
<abbr title="HyperText Markup Language">HTML</abbr>
```

---

## Insert Tag

The `<ins>` tag is used to show **inserted text** in a document.

### Example

```
<ins>Inserted text</ins>
```

---

## Delete Tag

The `<del>` tag is used to show **deleted text**.

### Example

```
<del>Deleted text</del>
```

---

## Strike Tag

The `<s>` tag is used to show text with a **line through it**.

### Example

```
<s>Striked text</s>
```

---

## Subscript Tag

The `<sub>` tag is used to display **subscript text**.

### Example

```
H<sub>2</sub>O
```

---

## Superscript Tag

The `<sup>` tag is used to display **superscript text**.

### Example

```
x<sup>2</sup>
```

---

## Anchor Tag

The `<a>` tag is used to create **links** to other webpages.

### Example

```
<a href="https://www.google.com/">Google</a>
```
